# css-state

## hover
```css
        .box:hover
        {
            background-color: green;

        }
```
- element에 마우스 커서가 올라가면 hover 속성이 적용된다.

## active
```css
.box:active
        {
            background-color: lawngreen;
            
        }
```
- 클릭하면 active 속성이 적용된다.

## focus
```css
.box:focus
        {
            background-color:blue;
            
        }
```
- 클릭하면 focus 속성이 적용된다.
