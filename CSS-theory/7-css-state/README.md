# css-state

## hover

- 마우스 커서가 올라가면 hover 속성이 적용된다.

```css
.box:hover {
  background-color: green;
}
```

## active

- 클릭하면 active 속성이 적용된다.

```css
.box:active {
  background-color: lawngreen;
}
```

## focus

- 클릭하면 focus 속성이 적용된다.

```css
.box:focus {
  background-color: blue;
}
```

## visited

```css
a:visited {
  color: blueviolet;
}
```

- 웹 페이지를 접속하면 하이퍼링크 색상이 변경된다.
